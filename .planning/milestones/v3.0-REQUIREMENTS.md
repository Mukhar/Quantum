## v3 Requirements (all validated)

### Qiskit-text export (Foundation)

- [x] **QSK-01** — User can click a "Copy as Qiskit" button on the
  sandbox toolbar to copy a runnable Qiskit Python snippet of the
  current circuit to the clipboard. Snippet uses
  `QuantumCircuit(n_qubits, n_qubits)`, applies each gate in `Step`
  order, includes measurement, and is verbatim-runnable in a Qiskit
  notebook. *(Validated by Phase 1, plan 01-02.)*
- [x] **QSK-02** — Every essay-embedded `CircuitView` exposes the
  same "Copy as Qiskit" affordance for its statically-encoded
  circuit. No essay's `CircuitView` is read-only. *(Validated by
  Phase 1, plan 01-03.)*
- [x] **QSK-03** — Drift-proof gate coverage: a golden unit test
  asserts every gate the simulator supports (`H`, `X`, `Y`, `Z`,
  `S`, `T`, `Rx`, `Ry`, `Rz`, `CNOT`, controlled rotations, `SWAP`,
  `M`) renders to a syntactically-correct Qiskit instruction. Test
  fails on any new simulator gate that lacks a Qiskit mapping.
  *(Validated by Phase 1, plan 01-01.)*

### Algorithm essays

- [x] **ALG-01** — `/teleportation` essay walks the reader through
  the 3-qubit teleportation circuit with an interactive
  `ProtocolStepper` widget. *(Validated by Phase 2, plans 02-01,
  02-02, 02-05.)*
- [x] **ALG-02** — `MultiBlochPanel` honestly renders mixed states:
  when a qubit's reduced density matrix has `|r| < 1`, the Bloch
  arrow renders **inside** the sphere (length = `|r|`).
  *(Validated by Phase 2, plans 02-01, 02-03, 02-05.)*
- [x] **ALG-03** — `/superdense-coding` essay shows the protocol
  with an `EncodingTable` widget. *(Validated by Phase 3, plans
  03-01, 03-02, 03-04.)*
- [x] **ALG-04** — `/grover` essay implements oracle + diffusion
  operators in the simulator with an `AmplitudeBars` per-iteration
  widget. *(Validated by Phase 4, plans 04-01, 04-02, 04-05.)*
- [x] **ALG-05** — `/shor` essay ships a `QFT` visualizer: a
  4-qubit QFT rendered as before/after probability bars over the
  computational basis, with reader-adjustable input state.
  *(Validated by Phase 5a, plans 05-01, 05-02, 05-04.)*
- [x] **ALG-06** — A `PeriodFinding` live demo runs in-browser for
  small modular-exponentiation periods (`a^x mod N` with `N ≤ 15`).
  *(Validated by Phase 5a, plans 05-01, 05-03, 05-05.)*
- [x] **ALG-07** — `/shor` essay ships a full N=15 Shor circuit
  rendered statically via `LargeCircuitView` (no in-browser
  execution) with the Qiskit-export button. Body text frames Qiskit
  as "now go run this for real." *(Validated by Phase 5b, plans
  05b-01, 05b-02, 05b-05.)*
- [x] **ALG-08** — `/vqe` essay ships a vanilla-TypeScript classical
  optimizer (~50-100 LOC gradient descent). Unit tests assert
  convergence on a 1D parabola and on the 2-parameter H2
  ground-state energy surface to within 1e-3 of the true minimum.
  *(Validated by Phase 6, plan 06-01.)*
- [x] **ALG-09** — An `EnergyLandscape` widget renders the
  variational energy surface for a 2-parameter ansatz; reader can
  drag a marker (or click "auto-descend") to watch the optimizer
  find the minimum. *(Validated by Phase 6, plans 06-03, 06-05.)*

### Use-case essays (one widget per half — no prose-only sections)

- [x] **USE-01** — `/teleportation` essay's second half ships a
  `QuantumNetwork` interactive: 3-node diagram (Alice / Repeater /
  Bob) where the reader clicks each link to swap entanglement.
  *(Validated by Phase 2, plans 02-04, 02-05.)*
- [x] **USE-02** — `/superdense-coding` essay's second half ships a
  `HolevoBound` widget that maps `n` qubits ↔ classical bits
  conveyable, surfacing the 2× advantage ceiling visually.
  *(Validated by Phase 3, plans 03-03, 03-04.)*
- [x] **USE-03** — `/grover` essay's second half ships a
  `SearchComparison` widget that side-by-side animates classical
  linear scan vs Grover's `√N` iterations as `N` slides from 16 to
  1024. *(Validated by Phase 4, plans 04-03, 04-05.)*
- [x] **USE-04** — `/shor` essay's second half ships an
  `RSACountdown` widget that takes RSA key size and a logical-qubit
  estimate slider, projecting the qubits-needed-to-break number and
  linking out to the 4 NIST PQC primitives. *(Validated by Phase
  5b, plans 05b-03, 05b-04, 05b-05.)*
- [x] **USE-05** — `/vqe` essay's second half ships a
  `MoleculeGallery` with 3 pre-baked molecules (`H2`, `LiH`, `HeH+`)
  — each shows the reader-saved ansatz parameters, the converged
  energy, and the Qiskit-export button. *(Validated by Phase 6,
  plans 06-02, 06-04, 06-05.)*
